Welcome! This is the official hands-on repo for the **Intro to GitHub** workshop.

## 🚀 Getting Started

### Step 1 — Fork this repo
Click the **Fork** button at the top-right of this page.  
This creates your own copy of the repo under your GitHub account.

### Step 2 — Clone your fork
```bash
git clone https://github.com/YOUR-USERNAME/intro-to-github.git
cd intro-to-github
```

### Step 3 — Follow the exercises
Head to the [`exercises/`](./exercises/) folder and start with `01-first-commit.md`.

---

## 🤝 Contributing (for participants)

1. Fork → Clone → Branch → Edit → Commit → Push → Pull Request
2. Your branch name should follow this format: `profile/your-name`
3. Only add files inside the `profiles/` folder — don't edit other people's files

---

## 📌 Cheat Sheet

```bash
git status                         # check what's changed
git add .                          # stage all changes
git commit -m "your message"       # commit with a message
git push                           # push to GitHub
git checkout -b branch-name        # create and switch to new branch
git pull                           # get latest changes
```
