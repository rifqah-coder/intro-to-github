# Contributing Guide

This repo is used for the **Intro to GitHub** workshop at UTP.

## Rules for participants

- Only create files inside `profiles/` — name it `your-name.md`
- Use the template provided in Exercise 1
- Branch name format: `profile/your-name`
- One PR per person
- Don't edit or delete other people's profile files

## Commit message format

```
add profile: your name
update profile: what you changed
fix profile: what you fixed
```

## Need help?

Raise your hand during the session or ping the facilitator on the group chat.
